Pregunta 1

El plan debe estar basado en un proyecto o en la organización completa. En el informe se debe detallar la información del proyecto para conocer el contexto en el cual se desarrolló el informe.

Respuesta: Verdadero

Pregunta 5

¿Qué es un cronograma?

  • Listado de tareas ordenadas de manera cronológica, necesarias para el logro de un objetivo o realización de un proyecto.
  • Listado para asignar recursos a las actividades para conocer el uso de estos recursos, administrándolos de la manera mas eficiente posible.
  • Listado de tareas ordenadas por prioridad, necesarias para el logro de un objetivo o realización de un proyecto.
  • Listado posee una representación gráfica de alguna actividades y nos permite conocer lo que se debera realizar, para controlar el proyecto.
Respuesta: Listado de tareas ordenadas de manera cronológica, necesarias para el logro de un objetivo o realización de un proyecto.

En la sección "Cronograma para el Plan de Gestión", el documento define: "Un cronograma es el listado de tareas ordenadas de manera cronológica, necesarias para el logro de un objetivo o realización de un proyecto."

Pregunta 8

La carta GANTT, a diferencia de PERT, es:

  • Un Diagrama de red.
  • Proporciona fechas limites para cada una de las actividades descritas.
  • Muy conveniente para proyectos en curso.
  • Muy conveniente para la etapa de planificación.
Respuesta: Muy conveniente para proyectos en curso.

En la sección "Diferencias entre Gantt y PERT", el documento indica que Gantt es "Muy conveniente para proyectos en curso", mientras que PERT es "Muy conveniente para la etapa de planificación".
